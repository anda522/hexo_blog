# 友链交换
想要交换友链的小伙伴，欢迎在**留言板**留言，留言格式：
* **名称：**你的博客名称
* **地址：**你的博客地址
* **简介：**一句话简介
* **头像：**你的头像地址

例如我的博客友链，大家可以加到自己博客里哦：
* **名称：**行码棋
* **地址：**http://wangyaqii.top
* **简介：**算法蒟蒻
* **头像**：http://wangyaqii.top/medias/logo.png
