## 1

第一个给了一个程序，进行复杂度分析

```cpp
int cal(int n)
{
	int x = 2, count = 0;
	while(x < n / 2)
	{
		x *= 2;
		count ++;
	}
	return count;
}
```
求count的值（用n表示）
求时间复杂度和空间复杂度
